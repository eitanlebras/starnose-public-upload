getting started

Start in fifteen seconds.

The daemon runs as a local HTTP proxy. Flip it on before a Claude Code session, flip it off after. Nothing persists in your environment between sessions.

1
npx starnose
install globally via npm
2
snose on
start proxy daemon on :3399
3
claude   # your normal workflow
every call intercepted automatically
4
snose off
stop daemon, env var cleared
~ terminal
$ snose on
✓ starnose daemon started on :3399
ANTHROPIC_BASE_URL=http://localhost:3399
recording to ~/.starnose/starnose.db
$ snose status
● running · 49 calls · session 2f8a · $1.24
$ snose off
